Тестовая работа для Astrio

Были реализованы все обязательные и необязательные задачи.

Реализована загрузка списка товаров и количество брендов через JSON.

Реализован функционал добавление товара в корзину, отображение кол-во добавленных товаров рядом с значком корзины.

Реализована фильтрация товаров через левый блок каталога. Все товары корректно отображаются только к тому бренду к которому они относятся.

Сделана адаптивность страниц до 250px, фреймворк при разработке адаптива не использовался, только чистый CSS.

Реализована страница корзины.

В странице корзины отображаются последние добавленные товары пользователем, добавленные товары сохраняются после перезапуска страницы (LocalStorage).

Реализовано изменение количества товара в корзине, возможность увеличить или уменьшить количество товара.

Реализован пересчёт итоговой стоимости всех товаров на основе кол-ва товара и их стоимости по отношению к друг другу в реальном времени.

Реализована возможность удалить товар из корзины.

Пользователь имеет возможность выбрать понравшиейся ему цвет товара, также его размер.

Товар в корзине отображается в контексте выбранного варианта. То есть если пользователь выбрал синие штаны с L размером, то отображаются именно они (фотография товара).

Выбранные варианты товара отображаются в корзине пользователя.

В случае, если у товара нету нужного цвета или размера, отображается его default вариант (Red/M размер)

Проект был создан с использованием Nuxt, для типизации использовался TypeScript, для стилизации использовался обычный CSS, без каких либо стилевых фреймворков или SASS/SCSS.

Проект логически разделён на компоненты, для взаимодействия компонентов между собой использовался стор Pinia

Для реализации возможности перехода на несколько страниц (Каталог и Корзина) использовался Vue Router.

Селекторы HTML и сопутствующие классы к ним были расписаны с использоваем методологии БЭМ

Используется кастомный шрифт Inter.ttf для придачи текстовому содержанию лучшего вида

Для стилизации блоков в HTML (Vue Template) использовалась система Flex и Grid

Для запуска достаточно лишь скачать репозиторий, установить все необходимые зависимости проекта и запустить его локальную среду (npm run dev)
